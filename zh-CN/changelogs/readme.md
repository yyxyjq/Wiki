## 2019

### 2019-12-29
sc开启了智能实体限制(yasui): 
- TPS过低时, 为了保证基本游戏不受影响（例如展开鞘翅等）, 服务器将会启用智能实体限制:  
    - 生物AI将会被抑制（部分怪物不会攻击、动物不会繁殖等）
    - 随机刻会被减少(植物不会生长，草地不会扩展)
    - 红石可能会被暂停
- 这些影响不是持续性的，正常情况下大约两三分钟会恢复正常
